# 🤖 Aura AI Assistant

Aura AI Assistant is a modern AI-powered chatbot built using **Flask**, **Groq Llama 3.3 70B**, **SQLite**, **HTML**, **CSS**, and **JavaScript**. It provides intelligent conversations, document analysis, chat history management, voice input, bookmarks, and a clean responsive interface.

---

## ✨ Features

- 💬 AI Chat using Groq Llama 3.3 70B
- 📄 Upload PDF, DOCX, TXT and Images
- 📑 Document Summarization
- 🧠 Chat History
- 🔍 Search Previous Chats
- 📌 Bookmarks
- 🎤 Voice Input
- 📋 Copy AI Responses
- 📤 Export Chats
- 🌙 Dark Theme
- ⚙️ AI Settings Panel
- 📝 Rename Chats
- 🗑 Delete Chats
- 📱 Responsive UI

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Aura-AI.git
```

Go to project

```bash
cd Aura-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```
GROQ_API_KEY=your_api_key
GOOGLE_API_KEY=your_google_api_key
```

Run the project

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
